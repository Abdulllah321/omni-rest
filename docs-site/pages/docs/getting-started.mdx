# Getting Started

Get up and running with Omni Rest in 5 minutes.

## Prerequisites

- Node.js 18+
- npm or yarn
- A Prisma project

## Installation

```bash
npm install omni-rest @prisma/client
npm install -D prisma
```

## 1. Define Your Schema

Create a `prisma/schema.prisma` file:

```prisma
generator client {
  provider = "prisma-client-js"
}

datasource db {
  provider = "sqlite"
  url      = "file:./dev.db"
}

model Product {
  id        Int      @id @default(autoincrement())
  name      String   @unique
  price     Float
  inventory Int      @default(0)
  createdAt DateTime @default(now())
  updatedAt DateTime @updatedAt
}

model Category {
  id    Int     @id @default(autoincrement())
  name  String  @unique
  products Product[]
}
```

## 2. Generate Prisma Client

```bash
npx prisma generate
```

## 3. Create Your API

### Express.js

```typescript
// server.ts
import express from "express";
import { PrismaClient } from "@prisma/client";
import { expressAdapter } from "omni-rest/express";

const app = express();
const prisma = new PrismaClient();

app.use(express.json());
app.use("/api", expressAdapter(prisma));

app.listen(3000, () => {
  console.log("🚀 Server running on http://localhost:3000");
  console.log("📖 API docs at http://localhost:3000/api/docs");
});
```

### Next.js App Router

```typescript
// app/api/[...prismaRest]/route.ts
import { PrismaClient } from "@prisma/client";
import { nextjsAdapter } from "omni-rest/nextjs";

const prisma = new PrismaClient();

export const { GET, POST, PUT, PATCH, DELETE } = nextjsAdapter(prisma);
```

### Fastify

```typescript
// server.ts
import Fastify from "fastify";
import { PrismaClient } from "@prisma/client";
import { fastifyAdapter } from "omni-rest/fastify";

const app = Fastify();
const prisma = new PrismaClient();

fastifyAdapter(app, prisma, {
  prefix: "/api",
});

app.listen({ port: 3000 });
```

## 4. Test Your API

Start your server and test the endpoints:

```bash
# Create a product
curl -X POST http://localhost:3000/api/product \\
  -H "Content-Type: application/json" \\
  -d '{"name":"Laptop","price":999.99,"inventory":10}'

# Get all products
curl http://localhost:3000/api/product

# Get a specific product
curl http://localhost:3000/api/product/1

# Update a product
curl -X PATCH http://localhost:3000/api/product/1 \\
  -H "Content-Type: application/json" \\
  -d '{"price":899.99}'

# Delete a product
curl -X DELETE http://localhost:3000/api/product/1
```

## 5. View API Documentation

Visit `http://localhost:3000/api/docs` to see the auto-generated Swagger UI documentation.
