# Getting Started

Get up and running with Omni Rest in 5 minutes.

## Prerequisites

- Node.js 18+
- A Prisma project

<Steps>

### Installation

Install the required packages using your preferred package manager.

<Tabs items={['npm', 'pnpm', 'yarn']}>
  <Tabs.Tab>
    ```bash
    npm install omni-rest @prisma/client
    npm install -D prisma
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash
    pnpm add omni-rest @prisma/client
    pnpm add -D prisma
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash
    yarn add omni-rest @prisma/client
    yarn add -D prisma
    ```
  </Tabs.Tab>
</Tabs>

### Define Your Schema

Create a `prisma/schema.prisma` file with your models.

```prisma
generator client {
  provider = "prisma-client-js"
}

datasource db {
  provider = "sqlite"
  url      = "file:./dev.db"
}

model Product {
  id        Int      @id @default(autoincrement())
  name      String   @unique
  price     Float
  inventory Int      @default(0)
  createdAt DateTime @default(now())
  updatedAt DateTime @updatedAt
}

model Category {
  id    Int     @id @default(autoincrement())
  name  String  @unique
  products Product[]
}
```

### Generate Prisma Client

<Callout type="info">
  You must regenerate the Prisma client every time you modify your schema.
</Callout>

```bash
npx prisma generate
```

### Create Your API

Choose your preferred framework below to see how to initialize the API instance:

<Tabs items={['Express.js', 'Next.js App Router', 'Fastify']}>
  <Tabs.Tab>
    ```typescript
    // server.ts
    import express from "express";
    import { PrismaClient } from "@prisma/client";
    import { expressAdapter } from "omni-rest/express";

    const app = express();
    const prisma = new PrismaClient();

    app.use(express.json());
    app.use("/api", expressAdapter(prisma));

    app.listen(3000, () => {
      console.log("🚀 Server running on http://localhost:3000");
    });
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```typescript
    // app/api/[...prismaRest]/route.ts
    import { PrismaClient } from "@prisma/client";
    import { nextjsAdapter } from "omni-rest/nextjs";

    const prisma = new PrismaClient();

    export const { GET, POST, PUT, PATCH, DELETE } = nextjsAdapter(prisma);
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```typescript
    // server.ts
    import Fastify from "fastify";
    import { PrismaClient } from "@prisma/client";
    import { fastifyAdapter } from "omni-rest/fastify";

    const app = Fastify();
    const prisma = new PrismaClient();

    fastifyAdapter(app, prisma, {
      prefix: "/api",
    });

    app.listen({ port: 3000 });
    ```
  </Tabs.Tab>
</Tabs>

  </Tabs.Tab>
</Tabs>

### View API Documentation

Visit `http://localhost:3000/api/docs` in your browser to see the auto-generated Swagger UI documentation for all endpoints, perfectly typed according to your Prisma schema!
